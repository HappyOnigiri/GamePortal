# Ethereal Strings

エーテル状の弦を奏でる、インタラクティブなオーディオビジュアル・アート作品です。

## 特徴
- **インタラクティブな体験**: 画面をスワイプすることで、仮想的な弦を弾き、音と視覚的な変化を楽しむことができます。
- **カスタマイズ**: 弦の数、張力（Tension）、雰囲気（Atmosphere/Reverb）を調整可能です。
- **美麗なビジュアル**: キャンバスを使用した滑らかなアニメーションと、ガラスモーフィズムを採用したUI。

## 使用技術
- TypeScript
- Vite
- Canvas API
- Web Audio API
